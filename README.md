# formio-ionic4
Formio working with ionic4
https://ionicframework.com/docs/building/android#running-with-cordova

install ionic cli globally
```npm i -g ionic```

# How run the project
1. ```npm install```
2. ```npm run dev:build```
on ios
```ionic cordova run ios```
on android
```ionic cordova run android```
